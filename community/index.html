Join the Documentarian community

Documentarian is an open source project that anyone in the community can use, improve, and enjoy. We'd love you to join us! Here's a few ways to find out what's happening and get involved.

Learn and Connect

Using or want to use Documentarian? Find out more here:

  • Authoring Guide: Read the authoring guide to get started with the Documentarian modules. The guide includes tutorials, concepts, and practices you can use when writing and maintaining documentation.
  • GitHub Discussions: The project discussions on GitHub host announcements, Q&A, feature ideas, and more. Head here to talk with other users, developers, and authors about the project and documentation.

Develop and Contribute

If you want to get involved by contributing to Documentarian, join us here:

  • Contributor Guide: Read the contributing guide to get started contributing directly. The guide includes everything you need to know about building, testing, extending, and documenting the project.
  • GitHub Repository: GitHub is where the project is hosted and developed. This repository has the source files for the project and this documentation site. It's also where all of the work gets tracked.
